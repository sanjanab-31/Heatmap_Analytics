# Heatmap Backend

Backend API service for the Heatmap Analytics Platform.

## Current Phase

Phase 1 scaffold is complete.

## Setup

1. Open this folder in a terminal.
2. Install dependencies:
   - npm install
3. Copy environment template:
   - cp .env.example .env
4. Start development server:
   - npm run dev

## Notes

- API implementation will be added incrementally in upcoming phases.
- Authentication is intentionally out of scope for v1 (hackathon mode).
